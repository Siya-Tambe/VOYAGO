## Overview

**VOYAGO** is a fully responsive, multi-page travel agency website built as a frontend design sample. It showcases a clean, modern UI for exploring global destinations, booking tour packages, browsing travel guides, and learning about safety — all through an interactive and visually engaging interface.

> *"Leave no trace, take only memories."*

## Features

- **Hero Section** — Full-screen landing with call-to-action and showreel link
- **Tour Packages** — Featured tours (Bali, Machu Picchu, Santorini) with duration and pricing
- **Destinations Page** — Explore travel locations with rich visuals
- **Expert Guides** — Showcase of lead guides (mountaineers, medics, photographers, naturalists)
- **Travel Safety** — Dedicated safety page with tips and guidelines
- **Trip Calendar** — View upcoming departures and available dates
- **Photo Gallery** — Visual collection of destinations and experiences
- **Travel Guides** — Informational guides for planning trips
- **Trip Planning Form** — Interactive inquiry form for custom trip planning
- **Contact Page** — Get in touch with the team
- **Terms & Waivers** — Legal and policy information
- **About Page** — Agency story and mission
- **Fully Responsive** — Optimized for desktop, tablet, and mobile
- **Smooth Navigation** — Mobile hamburger menu with clean UX

## Getting Started

### Option 1 — Open Directly (No Setup Required)

```bash
# Clone the repository
git clone https://github.com/Siya-Tambe/VOYAGO.git

# Navigate into the project
cd VOYAGO

# Open in your browser
open index.html
```

### Option 2 — Use a Local Server (Recommended)

Using VS Code's **Live Server** extension:
1. Open the project folder in VS Code
2. Right-click `index.html` → **Open with Live Server**

Using Python:
```bash
# Python 3
python -m http.server 8000

# Then visit http://localhost:8000
```

Using Node.js (`npx serve`):
```bash
npx serve .
```

> For detailed setup instructions, see [INSTALLATION.md](./INSTALLATION.md)
